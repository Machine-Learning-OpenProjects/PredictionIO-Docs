#######################
Welcome to PredictionIO
#######################

PredictionIO is an open source prediction server. 
It a powerful and flexible framework for programmers and data engineers to build smarter softwares with data science and machine learning algorithms.

By selecting one of the proven and scalable algorithms included in PredictionIO, web and mobile developers can
build many cool predictive features in minutes. For example, you can:

* offer personalized video, news, deals, ad and photo
* help users to discover interesting events, documents, apps and restaurants
* provide impressive match-making services
 
PredictionIO is built on top of state-of-the-art open source technology itself. 
We support Hadoop, Mahout, Cascading and Scalding natively within PredictionIO.

Community
*********

We want to ensure that PredictionIO is useful for you. You will get quick response to any question or feedback about PredictionIO through these channels: 

*GOOGLE GROUP*

This group (http://groups.google.com/group/predictionio-user) is actively supported by core contributors of PredictionIO. 
It is for general questions, requests for help, and new feature announcements.

*STACK OVERFLOW*

Stack Overflow is another great place to ask technical questions about PredictionIO. Be sure to add the predictionio tag to your question.

*GITHUB AND JIRA*

The code is on GitHub. The best way to send a patch is with a GitHub pull request. PredictionIO JIRA best way to file a bug report.
